## Content Management System

### Directus CMS Features

<AccordionGroup>
  <Accordion title="🎨 PPG Branded Interface">
    The admin interface is customized with PPG branding:
    - Custom green color scheme matching PPG identity
    - PPG logo integration throughout the interface
    - Consistent typography and visual hierarchy
    - Accessibility-compliant design elements
    
    <img src="/images/overview/branded-interface.png" alt="Directus admin interface with PPG green branding" />
  </Accordion>

  <Accordion title="👥 Role-Based Access Control">
    Four distinct user roles with specific permissions:
    
    **Content Creator**
    - Create and edit own content
    - Upload and manage own files
    - Submit content for review
    
    **Content Reviewer**
    - Review submitted content
    - Add comments and feedback
    - Approve or reject submissions
    
    **Content Publisher**
    - Publish approved content
    - Manage site settings
    - Configure navigation menus
    
    **Administrator**
    - Full system access
    - User management
    - System configuration
  </Accordion>

  <Accordion title="🔄 Content Workflow">
    Structured content approval process:
    
    ```mermaid
    graph LR
        A[Draft] --> B[In Review]
        B --> C[Approved]
        B --> D[Rejected]
        C --> E[Published]
        D --> A
        E --> F[Live on Website]
    ```
    
    - **Draft**: Work in progress, private to creator
    - **In Review**: Submitted for approval, reviewers notified
    - **Approved**: Ready for publication
    - **Published**: Live on the website
    - **Rejected**: Returned to creator with feedback
  </Accordion>

  <Accordion title="📊 Dashboard & Analytics">
    Comprehensive dashboard with real-time insights:
    - Content overview and status breakdown
    - Recent activity feed
    - Upcoming events calendar
    - Performance metrics
    - File storage usage
    - User activity tracking
    
    <img src="/images/overview/dashboard-panels.png" alt="Dashboard showing various analytics panels" />
  </Accordion>
</AccordionGroup>

## Integration Capabilities

### API Access

<AccordionGroup>
  <Accordion title="🔌 REST API">
    Full RESTful API for all content types:
    ```bash
    GET /items/blog_posts
    POST /items/blog_posts
    PATCH /items/blog_posts/{id}
    DELETE /items/blog_posts/{id}
    ```
    
    Features:
    - Filtering and sorting
    - Pagination support
    - Field selection
    - Relationship expansion
  </Accordion>

  <Accordion title="📊 GraphQL API">
    Flexible GraphQL endpoint for complex queries:
    ```graphql
    query {
      blog_posts(filter: {status: {_eq: "published"}}) {
        id
        title
        content
        featured_image {
          id
          filename_download
        }
      }
    }
    ```
    
    Features:
    - Single endpoint for all data
    - Precise data fetching
    - Real-time subscriptions
    - Schema introspection
  </Accordion>

  <Accordion title="🔗 Webhooks">

    - Custom trigger conditions
    
    Use cases:
    - Automatic site rebuilds

    - Notification systems
